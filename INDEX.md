# Skill Library Index

> Last updated: 2026-03-09

## Structure

Each skill is a directory with a `SKILL.md` as the entrypoint:

```
<skill-name>/
└── SKILL.md    # frontmatter + instructions
```

Place this repo at `~/.claude/skills/` to make all skills available across projects.
