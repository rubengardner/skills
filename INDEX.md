## Skill File Convention

Every skill follows the pattern:

```
Identity → Router → Specialists
```

- **Identity**: What this skill IS and who it serves.
- **Router**: Decision logic — which specialist handles which input.
- **Specialists**: Deterministic sub-instructions per scenario.

Each file is named: `SKILL_<NAME>.md`
